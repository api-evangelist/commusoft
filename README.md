# Commusoft

Commusoft is a cloud-based field service management software platform founded in 2006, purpose-built for trades businesses including HVAC, plumbing, electrical, and building maintenance contractors. The platform provides a comprehensive REST API that enables developers and integration partners to manage jobs, customers, engineers, quotes, invoices, parts, and service histories programmatically.
